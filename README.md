# 💬 Sentiment Analysis Dashboard
A data science project that cleans Twitter data and visualizes sentiment in a live dashboard.

## 🌐 Live Demo
👉 [View Dashboard](https://sentiment-dashboard-vnqk7j5bspmcxditu5sztx.streamlit.app)

## 📊 Features
- ✅ Cleans raw messy Twitter data
- ✅ Detects Positive, Negative, Neutral sentiments
- ✅ Interactive pie and bar charts
- ✅ Sentiment trends by Hour, Day and Month

## 🛠️ Tech Stack
| Tool | Purpose |
|---|---|
| Python | Core language |
| Pandas | Data cleaning |
| VADER | Sentiment analysis |
| Streamlit | Dashboard |
| Plotly | Charts |

## 🚀 Run Locally
```bash
pip install -r requirements.txt
python clean.py
streamlit run app.py
```
